# Boonthoop

A calm family home landing page built with Next.js and prepared for Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

For a production deployment:

```bash
vercel --prod
```
